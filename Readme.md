# How to use:

Divides files by a limit into need folders

## Project Stack

- nodemon

## Project setup
```
npm install
```

Paste all files (or folder link) into input.

In index.js determine in "const input" and the max amount of files per folder in "const dirLimit"

That is it.

## Deployment

**Watch:**

```bash
npm start
```

**Execute:**

```bash
node app.js
```
